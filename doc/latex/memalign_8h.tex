\subsubsection{\setlength{\rightskip}{0pt plus 5cm}{\bf mblock}$\ast$ memalign (size\_\-t {\em boundary}, size\_\-t {\em size})}}
\label{memalign_8h_1d2f6e98bba81d39e898911619bc99ce}


Allocate a block of memory with the malloc function and returns a \hyperlink{structmblock}{mblock}.

The \hyperlink{structmblock_b3a1756a7c6a5ede7265b6f025ae1dc5}{mblock::addr} field of the \hyperlink{structmblock}{mblock} returned is the address of a buffer aligned to a multiple of boundary.

The length of the aligned buffer is size.

To deallocate a \hyperlink{structmblock}{mblock} and the corresponding buffer, you must use the free function on the \hyperlink{structmblock}{mblock}. \begin{Desc}
\item[Parameters: ]\par
\begin{description}
\item[{\em 
boundary}]power of two that specifies the alignment of the buffer \item[{\em 
size}]size of the allocated buffer \end{description}
\end{Desc}
