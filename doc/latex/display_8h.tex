\subsubsection{\setlength{\rightskip}{0pt plus 5cm}void wait\_\-display\_\-refresh ()}}
\label{display_8h_5aaa4d219fc59ca1e37606cc2f17d306}


Wait for the refresh of the current \hyperlink{structdisplay}{display} to be completed. If there is no active \hyperlink{structdisplay}{display}, this will enter a deadlock.

This synchronisation mechanism is useful if you use functions that manipulate the active \hyperlink{structdisplay}{display} (especially \hyperlink{sprite_8h_dee6bffa6fb009e08ce22e260271ddee}{sort\_\-display\_\-layer}).

For this synchronisation mechanism to really work, you should have first initialised interrupts with \hyperlink{interrupt_8h_88ada49c1954d38252049786595d9a5f}{init\_\-interrupts} since it uses the \hyperlink{interrupt_8h_0f959e68baaa86d1982db2bcaf7b6a08}{vblCounter} variable.
